Create Open Graph Image for StoryStack

This tool helps you create a 1200x630 Open Graph image. Right-click the preview below and save it, or use the download button.

S

StoryStack

storystackstudios.com

Instructions:

  1. Click "Download as PNG" or right-click the preview and "Save Image As"
  2. Save the image as og-image.png in the apps/web/public/ folder
  3. The image should be exactly 1200x630 pixels
  4. After saving, commit and push the file to your repository

Alternative: Use online tools like Canva or OG Image Generator to create a custom design.
